Description

暨大论坛为Java开发的web项目，是一个Java学习论坛，为Java学习者提供一个学习的社区化平台。主要功能有登录、注册、浏览话题、发表话题、上传照片 话题分类浏览、评论、个人用户的积分、点赞（置为有用评论）、站内信通知、用户积分排行榜等等。它是一个校内讨论交流的平台，师生可以畅所欲言。

Technologies

Java 8 Standard Edition、 Spring MVC、 Spring Data、 Spring Security、 Spring Boot、 MyBatis、 HTML（Thymeleaf）、 CSS（Materialize）、 Maven、 MySql、 Redis

Software tools

Eclipse、 MySQLWorkbench [MySQL]、PostMan
